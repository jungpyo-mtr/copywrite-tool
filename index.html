사용법 — 4단계
STEP 01
폼 작성
아래 폼 6섹션 채우기. 카피 유형 + 사전 4질문 필수, 나머지는 선택.
→
STEP 02
프롬프트 복사
우측에 자동 생성된 텍스트 → 하단 "복사" 버튼 클릭.
→
STEP 03
Claude에 붙여넣기
claude.ai 웹 또는 Claude Code 대화창에 그대로 붙여넣기.
→
STEP 04
결과 검토
3안 + 근거 + 추천안이 포함된 카피 제안서를 받아 검토.
Claude 계정이 없다면 → claude.ai에서 이메일로 무료 가입 (1분). 별도 결제 없이 사용 가능합니다.
01 카피 유형 필수
작성하려는 카피의 유형을 하나 선택. 유형에 따라 우선 적용되는 작법 원칙이 달라집니다.
02 사전 4질문 필수
이 브랜드가 사라지면 무엇이 사라지나? "○○를 판다"가 아니라 "어떤 의미·관계·시간이 빠진다"까지.
추상어 감지: 더 구체화 필요. "신뢰감", "고급" 같은 표현 대신 실제 장면·관계·시간으로 표현.
누가, 어디서, 어떤 직전 행동 후, 어떤 마음으로 이 카피를 마주하나?
추상어 감지: 인구통계만으로는 부족. 검색어·직전 행동·심리 상태까지 포함.
단 하나만 전한다면 무엇을 전할 것인가? 한 카피 한 메시지 원칙.
추상어 감지: "최고", "혁신", "특별" 같은 추상어 대신 구체적 약속·장면으로.
어조와 한계 조건. 네 항목 모두 채우는 게 권장.
03 매체 캠페인·제품·바디 시
카피가 노출될 매체. 매체에 따라 호흡·길이·톤이 조정됩니다.
04 선택 입력 선택 — 있을 때 결과 품질 상승
경쟁사 카피 3~5개
브랜드 보이스 샘플 3~5개
레퍼런스 카피 (좋아하는 결)
피해야 할 방향
05 외부 검색 선택
06 출력 옵션 기본값 그대로도 OK
